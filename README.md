# Guess the number

It is a simple console based game.
In this game, the user have to guess the number from 0 to 30. There are lives given to the player i.e 5 lives here.
At the end, The player is asked if they want to continue the game or end the game.
When the number guessed by the player is greater or smaller than the random number generated, it say "Lower" or "Higher" respectively.
